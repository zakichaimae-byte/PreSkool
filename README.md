## 🌌 Vision & Esthétique "Deep Space"
**PreSkool** n'est pas seulement un outil de gestion, c'est une expérience immersive. L'application arbore un thème **"Deep Space Midnight"**, offrant un contraste élevé, des ombres douces (Glassmorphism) et une clarté visuelle exceptionnelle. Conçue pour être à la fois intuitive pour les étudiants et puissante pour les administrateurs.

---

## 🔥 Fonctionnalités Avancées

### 🏢 Architecture Multi-Espaces
-   **💎 Portail Administrateur** : Un centre de commandement pour piloter l'établissement. Statistiques en temps réel, graphiques circulaires pour la parité, et gestion centralisée des départements.
-   **🎓 Espace Étudiant Personnalisé** : Tableau de bord dynamique avec photo de profil intelligente (Avatars H/F), suivi des matières, emplois du temps hebdomadaires et calendrier des examens.
-   **👨‍🏫 Espace Enseignant Expert** : Gestion des présences par classe, saisie des notes avec coefficients, et accès rapide aux ressources pédagogiques.

### 🧩 Modules Intelligents
-   **📅 Gestion Académique** : Planification automatique des emplois du temps et des sessions d'examens.
-   **📑 Finance & Library** : Suivi rigoureux des frais de scolarité et gestion des stocks de la bibliothèque.
-   **🛎️ Notification & Événements** : Système d'alerte pour les vacances et les événements scolaires à venir.
-   **🌍 Localisation i18n** : Entièrement traduit en Français avec une gestion intelligente des locales pour une portée internationale.

## 🛠️ Stack Technologique & Excellence
L'application repose sur des technologies de pointe garantissant stabilité et performance :
-   **Core Engine** : Django 6.x (Framework Python haute performance).
-   **UI/UX Framework** : Bootstrap 5 + Vanilla CSS Custom Properties pour une réactivité fluide.
-   **Database Layer** : SQLite optimisé avec indexation des clés étrangères pour une intégrité absolue des données.
-   **Security** : Middleware d'authentification robuste avec filtrage des accès par groupe (Etudiant, Enseignant, Admin).

## 🏗️ Installation Rapide

1.  **Récupération du Code** :
    ```bash
    git clone [url-du-repo]
    cd PreSkool
    ```

2.  **Configuration de l'Environnement** :
    ```bash
    python -m venv venv
    .\venv\Scripts\activate  # Windows
    # source venv/bin/activate (Linux/Mac)
    ```

3.  **Déploiement des Dépendances & Base** :
    ```bash
    pip install -r requirements.txt
    python manage.py migrate
    ```

4.  **Décollage** :
    ```bash
    python manage.py runserver
    ```
